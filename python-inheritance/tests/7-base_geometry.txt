============================================
Comment utiliser 7-base_geometry.py
============================================

Cette bibliothèque est très simple, car elle ne contient qu'une seule fonction appelée
``integer_validator()``.

Nombres :
========
``integer_validator()`` retourne True si un nombre est un entier.
:: avec des entiers positifs
   >>> BaseGeometry = __import__('7-base_geometry').BaseGeometry
   >>> bg = BaseGeometry()

:: avec un entier positif
   >>> bg.integer_validator("my_int", 12)

:: avec un entier négatif
   >>> bg.integer_validator("number", -12)
   Traceback (most recent call last):
   ValueError: number must be greater than 0

:: avec 3 arguments
   >>> bg.integer_validator("numbers", 0, 1)
   Traceback (most recent call last):
   TypeError: integer_validator() takes 3 positional arguments but 4 were given

:: avec un entier infini
   >>> bg.integer_validator("infnumber", int('inf'))
   Traceback (most recent call last):
   ValueError: invalid literal for int() with base 10: 'inf'
